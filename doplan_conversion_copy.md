## OVERALL DESIGN SYSTEM NOTES

### Color Palette
- **Primary Accent:** Electric blue or cyan (e.g., #00D9FF or #0099FF)
- **Secondary Accent:** Warm complement (e.g., #FFB84D or emerald green)
- **Neutral Background:** Very dark (almost black, #0A0E27 or similar)
- **Neutral Text:** White (#FFFFFF) on dark, dark gray (#1A1F3A) on light
- **Accent Tints:** 10%, 20%, 30% opacity versions of primary accent for backgrounds

### Typography
- **Headlines:** Bold sans-serif (Roboto, Inter, Helvetica Neue)
- **Body:** Regular sans-serif (same family as headlines)
- **Monospace/Code:** For commands and code blocks (Monaco, Courier New)
- **Sizes:** Follow 1.5x scale (12px, 16px, 20px, 24px, 32px, 48px, 64px)

### Icon Strategy (from svgrepo.com)
- All icons: Same weight/style (outline or solid, not mixed)
- Size: 16px (micro), 20px (small), 24px (regular), 40px (large), 60px (hero)
- Color: Monochrome with accent color highlights
- Animation: Smooth, not jarring (200-400ms transitions)

### Animation Principles
- **On Scroll:** Fade in + slide up/down (300-500ms)
- **On Hover:** Color change + subtle scale (150-200ms)
- **On Load:** Staggered cascades (100-150ms between elements)
- **Continuous:** Slow, breathing animations (3-5s loops)
- **Respect prefers-reduced-motion:** Disable animations if user preference set

### Spacing
- **Section margins:** 60px (desktop), 40px (tablet), 30px (mobile)
- **Element padding:** 20px default (adjust for context)
- **Icon + text gap:** 12px (small), 16px (medium), 24px (large)
