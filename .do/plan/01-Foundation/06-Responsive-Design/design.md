## Breakpoint System

### Tailwind Breakpoints

```javascript
// tailwind.config.js
screens: {
  'sm': '480px',   // Small mobile
  'md': '768px',   // Tablet
  'lg': '1024px',  // Desktop
  'xl': '1440px',  // Large desktop
}
```

### Usage Pattern

```css
/* Mobile-first: Base styles for mobile */
.component {
  padding: 16px;
  font-size: 14px;
}

/* Tablet: md: prefix */
@media (min-width: 768px) {
  .component {
    padding: 24px;
    font-size: 16px;
  }
}

/* Desktop: lg: prefix */
@media (min-width: 1024px) {
  .component {
    padding: 32px;
    font-size: 18px;
  }
}
```

## Common Responsive Issues & Fixes

### Horizontal Scrolling

```css
/* Fix: Ensure no element exceeds viewport */
.container {
  max-width: 100%;
  overflow-x: hidden;
}

/* Fix: Use viewport units carefully */
.component {
  width: 100vw; /* Can cause horizontal scroll */
  width: 100%;  /* Better */
}
```

### Text Overflow

```css
/* Fix: Prevent text overflow */
.text {
  word-wrap: break-word;
  overflow-wrap: break-word;
  hyphens: auto;
}
```

### Image Responsiveness

```css
/* Fix: Make images responsive */
.image {
  max-width: 100%;
  height: auto;
}
```
