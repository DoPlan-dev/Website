## Usage

These rules are automatically loaded by your IDE's AI assistant. The rules are organized hierarchically and cover:

- Coding standards and best practices
- Framework-specific patterns
- Security guidelines
- Testing strategies
- Documentation standards
- And much more!

## Referencing Rules

To reference a specific rule in your commands or agent prompts, use:

```
@library/03-languages/go/go.md
@library/04-frameworks/frontend/nextjs.md
@library/09-devops-ci-cd/github-actions.md
```
