### System 3: Command Workflows & Subcommands âš¡

**Core Commands:**

`/hey` - Onboarding & guidance
- Subcommands: `meeting`, `plan`, `build`, `github`
- Interactive tutorial for new users
- Sets experience level, preferences, learning goals

`/do` - Start development phase
- Subcommands: `ideation`, `meeting`, `refining`
- Tracks time spent on tasks
- Creates Git branches for isolation

`/plan` - Scaffold project structure
- Subcommands: `phases`, `everything`
- Generates phase folders (01-foundation, 02-core_features)
- Creates feature templates and contracts

`/dev` - Development workspace
- Automatically creates branches
- Tracks time and progress
- Provides contextual guidance

`/sys` - System utilities
- `performance`: View metrics and optimization stats
- `engagement`: View achievements and scores
- `backup`: Backup projects and settings
- `restore`: Restore from backups

- 80-90% faster for new projects
