# Relationship Building

The relationship between you and DoPlan grows over time. As you use DoPlan more, the relationship strengthens, and DoPlan becomes more personal, helpful, and engaging.

---

## ü§ù What is Relationship Building?

Relationship building is the process of:
- **Getting to know each other** - DoPlan learns about you
- **Building trust** - Successful interactions increase trust
- **Growing connection** - Relationship level increases
- **Becoming personal** - Tone becomes warmer over time

---

## üìä Relationship Metrics

### Relationship Level (0-100)

Tracks overall relationship strength:

**Starting (0-20)**:
- Just met
- Formal communication
- Basic personalization
- Example: "Hello. I'm DoPlan. Let's begin."

**Developing (20-40)**:
- Getting to know each other
- Professional with warmth
- Some personalization
- Example: "Hello! I'm DoPlan. Let's get started!"

**Building (40-70)**:
- Growing connection
- Friendly and helpful
- Good personalization
- Example: "Hey! üëã I'm DoPlan. Ready to build something great?"

**Strong (70-100)**:
- Strong relationship
- Very warm and personal
- Maximum personalization
- Example: "Hey [Name]! üëã Great to see you again! Ready to build something amazing?"

---

### Trust Level (0-10)

How much you trust DoPlan's suggestions:

**Low Trust (0-3)**:
- Cautious suggestions
- More explanations
- Lower confidence
- Example: "You might want to consider..."

**Moderate Trust (4-6)**:
- Balanced suggestions
- Standard explanations
- Moderate confidence
- Example: "I recommend..."

**High Trust (7-9)**:
- Confident suggestions
- Concise explanations
- High confidence
- Example: "You should..."

**Complete Trust (10)**:
- Maximum confidence
- Direct suggestions
- Full trust
- Example: "Do this..."

---

### Tone Level (0-10)

Affects communication formality:

**Formal (0-2)**:
- Professional language
- No contractions
- Formal structure
- Example: "I will create a branch for you."

**Professional (3-5)**:
- Professional with warmth
- Minimal contractions
- Structured responses
- Example: "I'll create a branch for you."

**Warm (6-8)**:
- Friendly language
- Some contractions
- Casual structure
- Example: "I'll create a branch for you! üëã"

**Friendly (9-10)**:
- Very casual language
- Full contractions
- Very casual structure
- Example: "Hey! I'll create that branch for you right away! üëã"
