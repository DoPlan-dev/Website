# Tech Exploration

Discover and learn new technologies while building projects. DoPlan tracks your tech exploration and rewards you for trying new tools.

---

## üéØ What is Tech Exploration?

Tech exploration is discovering and learning new technologies:
- **Trying new tools** - Experiment with technologies
- **Learning as you build** - Educational support
- **Building diverse experience** - Multiple technologies
- **Earning achievements** - Tech exploration rewards

---

## üîß How Tech Exploration Works

### Automatic Detection

When you use a new technology:
- **Detected automatically** - From project context
- **Tracked in Memory Card** - Tech stack updated
- **Explanations provided** - Educational support
- **Achievements possible** - Tech Explorer, Tech Master

### Tech Stack Tracking

DoPlan tracks:
- **Technologies used** - Your tech stack
- **New technologies** - Exploration
- **Preferences** - What you like
- **Learning progress** - Skill growth
