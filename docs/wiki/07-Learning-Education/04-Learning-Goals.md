## ğŸ“ Setting Learning Goals

### During Onboarding

```bash
/hey
```

**What to do**:
- Set experience level
- Mention learning goals
- Choose learning focus
- Update preferences

**Example**:
```
DoPlan: "What would you like to learn?"
You: "I want to learn TypeScript and Docker"
```

### Through Conversation

```bash
/do
```

**What to do**:
- Mention learning goals in conversation
- Update as you learn
- Add new goals
- Track progress

**Example**:
```
You: "I want to build a project to learn TypeScript"
DoPlan: "Great! I'll make sure to provide TypeScript-specific guidance 
         and explanations throughout the project."
```

  ]
}

## ğŸ¯ How DoPlan Helps

### Educational Explanations

When working on learning goals:
- **Detailed explanations** - Understand concepts
- **Best practices** - Industry standards
- **Resources** - Where to learn more
- **Guidance** - Step-by-step help

### Example: Learning TypeScript

```
DoPlan: "I see you're learning TypeScript! This task will help you practice. 
        Let me explain:
        
        TypeScript adds type safety to JavaScript. Benefits:
        - Catches errors before runtime
        - Better IDE support
        - Improved refactoring
        - Better documentation
        
        I'll provide TypeScript-specific guidance throughout this task. 
        This is a great learning opportunity!"
```

### Tech Exploration

When you use a new technology:
- **Explanation provided** - What it is
- **Why it's useful** - Benefits
- **Learning resources** - Where to learn
- **Best practices** - How to use

## ğŸ’¡ Learning Goal Tips

### Setting Goals

1. **Be specific** - "Learn TypeScript" not "Learn programming"
2. **Set realistic goals** - Achievable targets
3. **Update regularly** - Add new goals as you learn
4. **Track progress** - Monitor achievement

### Achieving Goals

1. **Use in projects** - Apply what you learn
2. **Ask for explanations** - Understand concepts
3. **Explore technologies** - Try new tools
4. **Track progress** - See growth

### Updating Goals

1. **Add new goals** - As you discover interests
2. **Mark completed** - Celebrate achievements
3. **Refine goals** - Make them more specific
4. **Track progress** - Monitor learning
