## ðŸš€ Development Setup

### Prerequisites

- **Go** >= 1.23.0
- **Git** - Version control
- **Node.js** >= 14.0.0 (for npx wrapper)

### Setup Steps

1. **Clone repository**:
   ```bash
   git clone https://github.com/DoPlan-dev/CLI.git
   cd CLI
   ```

2. **Install dependencies**:
   ```bash
   go mod download
   ```

3. **Build**:
   ```bash
   go build -o doplan ./cmd/doplan
   ```

4. **Test**:
   ```bash
   go test ./...
   ```
