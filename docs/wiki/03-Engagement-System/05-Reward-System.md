# Reward System

The Reward System manages how achievements and challenges are awarded, displayed, and celebrated. It ensures rewards are meaningful and motivating.

### Reward Types

1. **Achievements** - Milestones and accomplishments (10-1000+ points)
2. **Challenges** - High-scoring first-time tasks (300-2000 points)
3. **Score Milestones** - Special achievements for score milestones (100-10,000 points)

- Do anything special

**Just use DoPlan normally!**
