# Testing & User Guides

This folder contains testing guides and user documentation for various features and components.

## Contents

- **Test Guides**: Step-by-step testing instructions
- **Quick Tests**: Quick testing procedures
- **Feature Guides**: Guides for specific features

## Files

- `DOCUMENTATION_TEST_GUIDE.md` - Guide for testing documentation features
- `PLAYGROUND_TEST_GUIDE.md` - Guide for testing the playground feature
- `TOKEN_EDITOR_TEST_GUIDE.md` - Guide for testing the token editor
- `QUICK_TEST.md` - Quick testing procedures
- `RESPONSIVE_TEST.md` - Responsive design testing guide

## Usage

These guides are intended for:
- Developers testing new features
- QA team members performing testing
- Contributors verifying functionality
- Documentation of testing procedures
