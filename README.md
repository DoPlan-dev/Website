## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/DoPlan-dev/CLI.git
   cd CLI/website
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Open in browser**

   Visit [http://localhost:5173](http://localhost:5173) to see the site in action.

## Commands

DoPlan comes with a powerful set of commands to streamline your workflow:

- `/hey` - Start a natural language conversation with your project director
- `/do` - Execute a specific task or instruction immediately
- `/plan` - Generate comprehensive implementation plans and architecture
- `/dev` - Enter interactive development mode with live feedback
- `/sys` - Manage system operations, deployment, and infrastructure

_Check the [Documentation](/docs) for a complete reference._
